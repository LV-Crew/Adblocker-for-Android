LV-Crew  Adblocker

The LV-Crew Adblocker allows you to update your hosts file with a downloaded blacklist.
The blacklisted traffic is being redirected to a blockpage (or a URL/localhost).
Therefore tracking is being reduced to a minimum.
You are also being protected from spam ads and more.

Number domains blocked: Up to over 815,000
LV-Crew Adblocker is OpenSource and therefore 100% free!

Download

No more worries

Your hosts file is being updated every hour, if you wish. You can enable auto-update in the settings dialog.


Easy Integration

All you have to do is to start the program and hit "Update hosts file". That's all! The program will take care of the rest.

Additional features

Additional features include the possibility to overwrite the URLs where you collect the hosts files from, overwriting the redirect IP, as well as automatic updates.

Why Us?

LV-Crew Adblocker redirects the traffic to a blockpage. Due to this, you won't see any more "Page not found" errors. Furthermore,
LV-Crew Adblocker is the only hosts manager which allows automatic updates. We do even take care of file permissions!
Currently, there are 36.164 blocked domains in our hosts file.

And the best thing about LV-Crew Adblocker: It is Open Source!

Blockpage

All traffic is being redirected to our blockpage. Due to that, there won't be any more "Page not Found" errors, which occur with other solutions.

Automatic updates

After enabling it in the Settings dialog, automatic updates will fetch and update the hosts file every hour. No user interaction is required, and it's happening silently.

Hosts file editor

You can now edit your hosts file right from within
LV-Crew Adblocker. It is also possible to to use wordpad as an external editor, if you wish.
File permissions are being handled by us, you won't have to worry about them.
